Overview

This is an application designed to graph the Dow Jones closing price over the past century. The information is parsed from this text file - which is also linked to by through the Data link in the navigation above.

The first several lines of the text file are shown below. The data is ordered as follows:

[year][month][day],[price]
        000103,66.61
        000104,67.15
        000105,66.71
        000106,66.02
        000108,66.41
        000109,64.99
        000110,64.14

Unfortunately, only the last two digits of each year are given which creates a miniature "Y2K" issue. The original text file was found at http://www.analyzeindices.com/.

Code

The code for this application is open sourced and accessible here. Whenever a change setting change is made (either to the graph or for a specific plot), a request is made to the server. The server then uses the request to parse the data and returns the relevant result to the browser.

The data points are limited from the original 29356 to a number more capable of being rendered by the browser's JavaScript.

What can be done

Currently there are three main features. Firstly, a user can sample a desired number of data points. This is done through the slider on the main page. The sampled points render on the graph - along with the rendering times for both the browser and the server. It is these points that are used as the data for the graphs. The second feature is the ability to plot a Running Median of the data points selected by the user. The third feature is the ability to plot plotting a 3-4 Smooth of the data points selected by the user.

A Chebyshev Interpolation plot is currently implemented in an alpha form as well.

Technology used

This page is powered by Google App Engine. The interface uses jQuery, jQuery UI and Blueprint. The graph is powered by Flot. The code is open sourced at Google Code. The mpmath python library is packaged as well although not fully integrated and used.
